# 触发矩阵

| 触发时机 | 前置条件 | 执行动作 | 主要输出 |
|---|---|---|---|
| 新建或更新工单 | 客户描述变化 | 描述修正 | 修正描述、描述修正状态 |
| 新建或更新工单 | 附件变化 | 附件解析 | 附件状态、摘要、结构化结果、错误 |
| 中间字段更新 | 描述修正已完成，且附件已有明确状态 | 补充判断与预回复生成 | 缺失信息、内部判断依据、补充话术或对外预回复 |
| 员工更新修正内容 | 存在可供确认的回复 | 最终回复整理 | 最终回复草稿 |
| 员工点击确认复制 | 存在员工确认版本 | 复制内容并记录操作 | 确认复制人、时间、最终版本、状态 |

## 防重复约束

- 仅在字段值实际变化时触发。
- 补充判断与预回复生成必须等待描述修正和附件解析两个步骤完成。
- 确认复制按当前路径更新状态：补充话术进入“待补充信息”，正式回复进入“已完成”。
