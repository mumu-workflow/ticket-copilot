# 发布边界

## 可以公开

- 通用字段结构
- 状态机和触发矩阵
- 非敏感 Prompt 结构
- 合成脱敏案例
- 合成知识库和话术库样例

## 禁止公开

- 自动化 ID、字段 ID
- 账号、Token、密钥
- 内网地址
- 真实客户数据和附件
- 内部截图中的敏感信息
- 内部专有 Prompt 原文

## 版本规划

### v1.0.0

- 描述修正
- 附件解析
- 信息完整度判断
- 补充话术
- 对外预回复
- 人工确认复制
- 固定状态机

### v1.1.0

- 通用知识库接入
- 标准话术库
- 知识命中记录
- 教程链接引用
- 知识沉淀规则
