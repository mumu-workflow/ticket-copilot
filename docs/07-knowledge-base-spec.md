# 知识库规范

v1.0 可以不接入知识库。v1.1 再增加知识库和标准话术库能力。

## 推荐结构

| 字段 | 说明 |
|---|---|
| id | 公开样例中的唯一标识 |
| 问题类型 | 问题分类 |
| 触发关键词 | 检索和规则匹配用关键词 |
| 标准回复 | 对外可发送主体内容 |
| 适用场景 | 使用条件 |
| 限制条件 | 不适用边界 |
| 教程标题 | 教程名称 |
| 教程链接 | 可公开访问链接 |
| 是否已验证 | 是否经过人工确认 |
| 更新时间 | 最近维护时间 |

## 使用原则

1. 仅引用已验证知识。
2. 未命中时提示员工人工判断。
3. 只沉淀员工确认后的最终版本。
4. 不沉淀客户隐私、未经验证结论或单客户临时方案。

最小示例见 [知识库样例](../examples/kb/kb-minimum-10.json)。
