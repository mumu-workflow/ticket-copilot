# 字段规范

## 输入字段

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| 工单编号 | 文本 | 是 | 工单唯一标识 |
| 客户原始描述 | 多行文本 | 是 | 客户原始问题 |
| 附件 | 附件 | 否 | 图片或文档 |
| 附件提取文本 | 多行文本 | 否 | OCR 或解析结果 |
| 人工修正内容 | 多行文本 | 否 | 员工调整后的回复 |
| 人工修正思路 | 多行文本 | 否 | 员工修正依据 |

## 中间字段

| 字段 | 类型 | 说明 |
|---|---|---|
| AI 修正后的描述 | 多行文本 | 规范化描述，不新增事实 |
| 描述修正状态 | 单选 | 待处理 / 已完成 |
| 附件解析状态 | 单选 | 无附件 / 解析成功 / 格式不支持 / 解析失败 |
| 附件解析摘要 | 多行文本 | 面向员工的摘要 |
| 附件结构化结果 | JSON 文本 | 结构化证据 |
| 附件解析错误 | 单行文本 | 失败原因 |
| 是否需要补充 | 单选 | 是 / 否 |
| 缺失信息项 | 多选 | 缺少的必要资料 |
| 内部判断依据 | 多行文本 | 仅记录知识命中、规则命中和缺失信息项 |

## 输出字段

| 字段 | 类型 | 说明 |
|---|---|---|
| 补充信息话术 | 多行文本 | 信息不足时复制发送 |
| 对外预回复（可复制） | 多行文本 | 不包含模型思考过程 |
| 最终回复（员工确认版） | 多行文本 | 员工确认后的版本 |
| 确认复制人 | 成员 | 点击确认复制时记录 |
| 确认复制时间 | 日期时间 | 点击确认复制时记录 |
| 状态 | 单选 | 固定状态机 |
