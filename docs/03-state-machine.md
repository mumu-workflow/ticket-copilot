# 状态机

| 状态 | 进入条件 | 允许动作 | 退出条件 |
|---|---|---|---|
| 待分析 | 新建工单，或客户补充资料 | 描述修正、附件解析、信息判断 | 生成补充话术或对外预回复 |
| 待补充信息 | 客服确认复制补充话术 | 等待并录入客户补充资料 | 客户资料更新 |
| 待发送确认 | 信息充分且已生成对外预回复 | 客服查看、修正、确认复制 | 客服确认复制正式回复 |
| 已完成 | 客服确认复制正式回复 | 查看最终版本 | 流程结束 |

## 说明

- 不设置“待人工确认”：人工查看是所有回复的固定步骤。
- 不设置“待人工处理”：升级人工建议作为字段和回复内容，不新增状态。
- 系统无法确认外部工具中的真实发送结果，因此以“确认复制”作为流程内完成动作。
